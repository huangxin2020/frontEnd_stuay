## 什么是ES6
ECMAScript 6.0 （简称ES6） 是继ECMAScript 5.1以后的javascript 语言的下一代标准，在2015年6月份发布。
他的目标是使javascript语言可以用来编写复杂的大型应用程序，成为企业级开发语言。

其中前6部分为新增的基础语法。

1. let和const关键字
2. 变量的解构赋值
3. Set数据结构
4. Map数据结构
5. Iterator 和 for...of循环
6. Class基本语言

7. Symbol
8. 对象的扩展
9. 函数的扩展
